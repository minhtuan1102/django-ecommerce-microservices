## Discount Codes

### How to Use
1. Add items to cart
2. At checkout, enter code in "Discount Code" field
3. Click "Apply"
4. Discount reflects in order total

### Code Types
- **Percentage Off**: e.g., "SAVE20" for 20% off
- **Fixed Amount**: e.g., "50K" for 50,000 VND off
- **Free Shipping**: e.g., "FREESHIP"
- **Category-Specific**: e.g., "FICTION15" for 15% off fiction

### Common Issues
- **"Code Invalid"**: Check spelling or expiration date
- **"Not Applicable"**: Code may have restrictions (minimum purchase, specific categories)
- **"Already Used"**: Single-use codes can't be reused

## Referral Program

### How It Works
1. Share your unique referral code with friends
2. Friend gets 50,000 VND off their first order (min. 200,000 VND)
3. You get 50,000 VND credit when they complete purchase
4. Plus 200 bonus points!

### Find Your Code
Go to Account > Referrals to get your personalized code and tracking link.

### Terms
- New customers only
- One referral bonus per new customer
- Credits expire in 90 days
- No limit on number of referrals
